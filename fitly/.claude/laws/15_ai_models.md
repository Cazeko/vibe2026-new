# 법률 15 — AI 사용 규정

> 본 법률은 헌법 v3.3 제18조 (v3.1 매트릭스)·제18조의2의 본문이다.

## 제18조 (AI 사용 규정 — 모델 매트릭스) [v3.1 전면 개정]

1. **시스템(앱) 런타임의 LLM 호출은 Google Gemini 단일 공급자를 사용한다.** 주인님이 보유한 회사 키 단일성을 유지한다.
   - **단서 (v3.1 신설)**: 운영자가 1회성 시드 데이터 생성·태깅·모범답안 작성 단계에서 외부 LLM (Anthropic Claude 등)을 활용하는 경우, 본 항의 단일 공급자 원칙은 적용되지 아니한다. 시드된 데이터 자체는 모델 식별자를 포함하지 아니하므로 시스템 런타임의 단일 공급자 원칙과 충돌하지 아니한다. 외부 LLM 활용 시 사용자(주관자)의 사전 승인을 받는다.

2. 모델 정책 — **기능별 차등 매트릭스** [v3.1 — "성능 최우선·가성비 무관" v1.5 정책 폐기]
   - 본 매트릭스는 *작업의 성격*에 따라 모델을 선택한다. 추론 깊이가 핵심인 작업에는 Pro, 대량·단순 분류는 Flash, 한국어 서술 품질이 핵심인 시드 작업은 Sonnet 1M.
   - 비용을 무시하지 아니하되, *기능 적합성*을 우선한다.

   **A. 시스템 런타임 (앱이 사용자에게 응답할 때)**

   | 작업 | 모델 | 사유 |
   |---|---|---|
   | Vision OCR (시험지 사진/이미지 PDF) | `gemini-3.1-pro-preview` | 멀티모달 1M 입력, 표·이미지 보존 |
   | 학습 적합도 해설·코칭 답변 | `gemini-3.1-pro-preview` | 추론 품질 |
   | 팟캐스트 스크립트 생성 (NotebookLM 스타일) | `gemini-3.1-pro-preview` | 대화체·multi-speaker 호환 |
   | 팟캐스트 multi-speaker TTS | Gemini multi-speaker TTS | 한국어 화자 |
   | 가벼운 분류·태깅·요약 (대량 호출) | **`gemini-2.5-flash`** | 가성비 우선, 호출량 폭주 차단 |
   | 임베딩 (RAG, Phase 2+) | `gemini-embedding-2` | 신세대 3072차원 |

   **B. 시드 데이터 운영 (1회성·매년 갱신, 운영자 작업)**

   | 작업 | 단기 (현 시점) | 장기 (매년 갱신) | 사유 |
   |---|---|---|---|
   | 시험지 텍스트 → 문항 분리 (구조화 JSON) | **`claude-sonnet-4-6` (1M ctx)** | `gemini-3.1-pro-preview` | 단기는 Anthropic 키 활용, 장기 자동화는 Gemini |
   | 문항 영역/Bloom/형식 태깅 | `claude-sonnet-4-6` | **`gemini-2.5-flash`** | 장기는 대량 분류 → Flash 가성비 |
   | 키워드 태깅 + dedup | `claude-sonnet-4-6` | **`gemini-2.5-flash`** | 동일 |
   | 서술형 모범답안 생성 (한국어 4~8문단) | `claude-sonnet-4-6` | `gemini-3.1-pro-preview` | 한국어 학술 문체 품질 우선 |
   | 단기 prompt caching | Anthropic prompt caching 활용 (5분 TTL) | — | few-shot 컨텍스트 재사용으로 입력 비용 50~70% 절감 |

3. SRS 알고리즘은 **`ts-fsrs`** 라이브러리를 사용하며 자체 구현은 금한다.
4. PDF 파싱은 **`unpdf`** (텍스트 PDF) 및 위 매트릭스의 Vision OCR 모델 (이미지 PDF)을 표준으로 한다.
5. 모델 ID는 환경 변수로 노출하여 출시 주기에 따른 업그레이드를 키 교체 없이 수행 가능하게 한다.
   - `GEMINI_MODEL_PRO`, `GEMINI_MODEL_FLASH`, `GEMINI_EMBEDDING_MODEL`
   - `ANTHROPIC_MODEL_SONNET` (v3.1 신설 — 시드 운영용)
6. 본 매트릭스를 변경하려면 헌법 개정 절차를 거친다.
7. **비용·품질 검증 게이트 (v3.1 신설)** — 본 매트릭스의 "장기" 자동화 본실행 전, 5개 PDF 샘플로 토큰·달러 실측 후 사용자 사전 승인 (시드 파이프라인 plan §8 D-S6a 정합). 장기 매트릭스가 단기 결과물 대비 품질 회귀가 있을 경우 매트릭스 재조정 (헌법 개정).

## 제18조의2 (AI 생성 정답·해설의 라벨 의무)

1. AI(Gemini)가 추정한 정답·해설은 사용자에게 노출될 때 반드시 **"검증 필요"** 배지와 함께 표시한다.
2. 사용자 신고·정정에 의해 검증된 카드는 다음 단계의 라벨로 승격된다 (제30조의2 참조).
3. 라벨 없는 AI 정답을 표시하는 것은 본 헌법 위반이며, 발견 시 즉시 정정한다.

## 제18조의3 (객관식 시대 데이터 운영 — 연도 범위 명문화) [v3.5 신설]

1. **객관식 시대 = 2002학년도 ~ 2013학년도 12회분**으로 정의한다 (KICE 공식 시험 형식 변천 정합).
2. 객관식 시대 기출은 **풀이 카드(QuizCard) 출제 대상에서 제외**한다 (CHANGELOG v3.2 결정 명문화).
3. 객관식 시대 데이터는 **개념 정리 노트(KeywordCard)** 로 흡수하여 RAG·토픽맵·로드맵에 한해 활용한다.
4. 2014학년도부터는 **서술형 시대**로 분류하며, 풀이 출제 대상이 된다.
5. 2002학년도 이전(2001 이하) 자료는 시드에 포함하지 아니한다 (rules/33 제35조의2 정합).
