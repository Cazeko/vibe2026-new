# 법률 13 — 모듈 cut 결정 + 지역 라벨

> 본 법률은 헌법 v3.3 제14·15조의 본문이다.

## 제14조 (모듈 구성 — Cut 결정)

다음 기능은 본 대회 출품 범위에서 **제외**되었으며, 다시 도입하려면 헌법 개정이 필요하다.

1. 마인드맵 (Mermaid·Markdown 모두 cut)
2. 자유 노트 에디터
3. 카드의 *AI 무작위 생성* (시드 추출만 허용)
4. 카톡 봇 진입점 (3주 리스크)
5. **사용자 PDF 자료 업로드 + AI 카드 자동 추출** [v3.0.1 신설]
   - 사유: ① v3.0 컨셉 피벗 후 시드 1,000+ 카드(공식 KICE 기출)로 학습 콘텐츠 충분, ② 합법 시나리오는 "본인 정리 노트"에 한정되나 사용자가 PDF 변환 후 업로드하는 마찰이 헌법 제5조 진입장벽 0과 충돌, ③ 사설 학원 모의고사 업로드 시 제27조 저작권 위배 회색지대 발생, ④ v3.0 마법 지점은 시드 분석 즉시 노출 + 팟캐스트 자동 생성으로 재정의되어 사용자 업로드는 차별화·마법성 어디에도 속하지 않음.
   - 영향: `materials` 테이블·`/materials` 라우트·`/api/materials`·`src/lib/ocr/study-cards.ts`·`src/lib/ocr/mistake-cards.ts` 폐기 (D-S1.5 후속 처리).
   - 본 cut의 도입 재고려는 헌법 개정 절차 (제40~45조)에 따른다.

## 제15조 (지역 교육청 라벨 — v3.0 재정의)

1. 사용자는 응시 예정 **지역 교육청**을 선택 입력할 수 있다 (서울·경기·인천·부산·대구·광주·대전·울산·세종·강원·충북·충남·전북·전남·경북·경남·제주 17개).
2. 단, Fitly 는 어떠한 교육청에 대해서도 **합격 컷·평균·경쟁률 추정 시드를 보유하지 아니한다** (제3조의2 정합).
3. UI 에서 교육청별로 노출 가능한 정보는 **공개된 시험일·시험과목·시험시간·시험장소** 등으로 한정한다.
4. 본 조 입력은 선택 사항이며, 미입력 시에도 모든 학습 기능이 정상 동작한다 (시드 데이터는 전국 공통).
