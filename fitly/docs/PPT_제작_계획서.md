## 1. 디자인 시스템 (DESIGN.md §4 정합)

### 1.1 색 토큰

라이트 모드 기준 (PPT 는 라이트 톤 권장):

| 토큰 | HEX | 용도 |
|---|---|---|
| **cream** (배경) | `#F9F5EA` 또는 `#FAF6EE` | 모든 슬라이드 배경 |
| cream-soft | `#FFFAF1` | 카드 배경 |
| cream-deep | `#F3EDE0` | 강조 카드 배경 |
| **ink** (본문) | `#15172D` 또는 `#1A2027` | 헤드라인·본문 글자 |
| ink-soft | `#434B55` | 부드러운 본문 |
| muted | `#6B6256` 또는 `#8A8478` | 캡션·메타 |
| **evergreen** (액센트) | `#1F5C4A` 또는 `#2F5D50` | **단 1색만** — 강조 단어·KPI 숫자·rule |
| evergreen-strong | `#173F33` | hover·강조 강화 |
| rule | `#E8E2D5` | 가는 가로 구분선 |
| rule-strong | `#C8BFA8` | 카드 보더 |

> **핵심 규칙**: evergreen 은 **한 슬라이드에 한 단어**만 강조. 모든 단어를 evergreen 으로 칠하면 "한 번만 빛난다" 원칙 위배.

### 1.2 폰트

| 용도 | 폰트 | 굵기 |
|---|---|---|
| 헤드라인 | **Noto Serif KR** (또는 Source Han Serif K) | Black (900) |
| 부제·본문 | **Noto Sans KR** (또는 Pretendard) | Regular / Medium / Bold |
| 강조 캡션 | Noto Sans KR | Bold + tracking 4~6 |
| 인용·펀치라인 | Noto Serif KR | Italic + Bold |

### 1.3 폰트 크기 표준

| 요소 | 크기 |
|---|---|
| 슬라이드 제목 (헤드라인) | **46~52pt** |
| 표지 메인 ("Fitly") | **100~120pt** |
| 큰 KPI 숫자 | 36~52pt |
| 섹션 헤더 | 17~20pt |
| 본문 | 13~14pt |
| 캡션·메타 | 11~12pt (muted) |
| 푸터 | 10pt |

### 1.4 여백·카드 스타일

- 좌우 여백: **0.65"** (16.5mm)
- 상하 여백: **0.55"** (14mm)
- 카드 corner radius: **0 또는 2px max** (둥근 카드 X — 공책 정합)
- 보더: **1px** rule-strong (`#C8BFA8`)
- 그림자: **없음** (drop shadow X — 평면 종이 느낌)
- 강조 카드: 좌측 3px evergreen 보더 단일 (한쪽만)

### 1.5 종이 그레인 (선택)

DESIGN.md §4.5 정합. CSS `radial-gradient` 4px 노이즈 1~2% opacity. PPT 도구가 지원하면 배경 텍스처로 추가, 아니면 단색 cream 만으로도 OK.

## 2. 14 슬라이드 상세 계획

각 슬라이드마다:
- **목적** — 청중 머릿속에 남기고 싶은 한 가지
- **헤드라인** — 큰 세리프 한 문장
- **콘텐츠** — 본문 구성
- **시각 요소** — 추천 비주얼
- **발표 멘트** — 예시 (약 30~45초)
- **주의** — 빠지기 쉬운 함정

### 슬라이드 6 · PART 4 · 데이터 파이프라인 + 기술 스택

| 항목 | 내용 |
|---|---|
| 목적 | 기술 신뢰도 — "진짜 만든 팀" 인상 |

## 7. 자주 묻는 디테일

### Q. 슬라이드 11 unit economics 숫자는 어디서 나왔나요?
- **풀이 채점 1회 ≈ 5~10원**: Gemini Flash 단가 기준 (input ~$0.075/1M, output ~$0.30/1M) + 풀이 1회당 input 3K + output 2K tokens 추정
- **사용자당 월 비용 ≈ 1,200~3,000원**: Pass 활성 사용자 풀이 200~400회/월 + 팟캐스트 TTS 1~2회 추정
- **서버 고정비 ~60,000원/월**: Supabase Pro $25 + Vercel Pro $20 + 도메인 등
- **마진 70~90%**: 시즌 6개월 Pass 49,000원 대비 비용 6,000~15,000원
- 모두 *가설 추정치* — 실 운영 안 됐으므로 발표 멘트에서 "추정"·"가설" 명시

### Q. 슬라이드 6 보안 "9 user-scope 테이블" 근거?
- `supabase/migrations/` 디렉토리 SQL 마이그레이션 중 `enable row level security` 명시된 파일 9개
- 사용자 단위 격리 (user_card_state, user_card_log, user_profiles 등)
- 추가로 미들웨어(`src/lib/supabase/middleware.ts`)에서 OAuth PKCE callback 보호 + CSRF Origin 검증

### Q. 영상 슬라이드(8·9)에서 발표자는 뭘 하나요?
- 영상 재생 직전 약 5초 멘트 (소개)
- 재생 중에는 *말을 안 함* — 화면이 다 보여줌
- 재생 끝나면 다음 슬라이드 바로 넘김 ("이상 사용성 영상이었습니다" 같은 부속 멘트 X)

### Q. 슬라이드 13 회고가 너무 길지 않을까요?
- 30~50초 분량 권장
- 핵심: 종전(CLAUDE.md) → 전환(4 계층) → 결과(스코프 보호). 이 3 단계 서사를 한 호흡에
- *"한국 법체계처럼"* 비유가 가장 임팩트 강함 — 이 한 줄은 반드시 넣을 것

### Q. 도구별 추천?
- **Gamma AI**: outline 자동 변환 빠름, 톤 통일 어려움 (theme 커스터마이즈 한계)
- **Figma**: 픽셀 단위 통제 최강, 학습 곡선 있음
- **Keynote**: Mac 사용자, 톤 정합 쉬움, 영상 임베드 강력
- **PowerPoint**: 전 OS 호환, 익숙함, 글자 자동 늘어남 주의
