## 1. 발의 배경

2026-05-14 brainstorming 합의의 **AI 서술형 학습 워크스페이스 6 PR 시리즈** 의 마지막 단계인 **PR 6 (AI 학습 도우미)** 진입 시점에서, 다음의 정합 문제가 동시에 관찰되었다.

### 1.1 §16 (스코프 보호) 와의 *형식적* 충돌

PR 1~5 는 기존 풀이 트랙(quiz/mistake)의 자가 채점·정답 비교 기능을 *다듬는* 형태로 §16 단서(v3.5.3)로 흡수되었다. 다만 **PR 6 의 AssistantFab + 챗봇** 은 다음과 같은 *새로운 인터랙션 진입점* 을 도입한다.

- 우측 하단 플로팅 FAB (신규 UI 진입점)
- multi-turn 대화 (신규 LLM 호출 패턴)
- 빠른 프롬프트 5종 (재학습·약점 진단·유사 기출·암기 카드·심화 학습 안내)

이는 "기존 기능 다듬기" 단서(v3.5.3 UI 개선 단서)로는 흡수되기 어렵다. 따라서 §16 본문에 *학습 본업 AI 도우미* 단서를 명시하여 정합을 회복한다.

### 1.2 §18 매트릭스의 모델 ID 노후화

- §18 매트릭스 A (시스템 런타임) 의 "가벼운 분류·태깅·요약" 행은 `gemini-2.5-flash` 로 명시되어 있으나, PR 2 (2026-05-14 머지 #40) 에서 `env.modelFlash` fallback 을 `gemini-3.0-flash` 로 정합 갱신하였다. 본 발의로 매트릭스 ID 를 동기화한다.
- AI 서술형 분석(PR 2~5)·AI 학습 도우미 챗봇(PR 6) 은 매트릭스에 **신규 행 2건** 으로 명시되어야 §18-6 "본 매트릭스를 변경하려면 헌법 개정 절차를 거친다" 정합.

### 2.2 제안 조항

#### A. §16 단서 추가 (v3.6.1 — 헌법층)

기존 단서 뒤에 다음 단서를 *연속 추가* 한다.

> **단서 (v3.6.1 — 2026-05-14)**: 풀이 트랙(quiz/mistake) 의 **AI 학습 도우미** (플로팅 진입점·빠른 프롬프트·multi-turn 챗봇)는 PR 1~5 의 AI 서술형 분석 인프라(`user_card_ai_analysis` 캐시·Gemini Flash wrapper)의 *자연 확장* 으로 분류한다. 본 단서는 *학습 본업(풀이 트랙)에 한정* 되며, 학습 본업 외 영역(대시보드·기출분석·팟캐스트 등) 으로의 확장은 별도 헌법 개정을 요한다.

#### B. §18 A 매트릭스 — 기존 행 ID 갱신 + 신규 행 2건

**기존 행 ID 갱신** (§18-5 환경변수 노출 정합으로 자동 업그레이드 인정):
> | 가벼운 분류·태깅·요약 (대량 호출) | **`gemini-3.0-flash`** | 가성비 우선, 호출량 폭주 차단 |

**신규 행 2건 추가**:
> | 서술형 답안 AI 정성 분석 (강점·보완점·키워드·diff) | `gemini-3.0-flash` | 정성 피드백, attempt_hash 캐시로 LLM 재호출 0회 |
> | AI 학습 도우미 챗봇 (카드 컨텍스트 multi-turn) | `gemini-3.0-flash` | 대화형 보조·빠른 응답·세션 메모리 보존 |

### 2.3 개정 사유

1. **§16 단서** — PR 6 의 챗봇·FAB 는 PR 1~5 의 자연 확장이지만, *형식적* 충돌이 잔존. 단서로 명시 적용 범위를 *학습 본업* 으로 한정하여 *대시보드 챗봇·팟캐스트 챗봇 등 향후 스코프 폭주* 를 헌법 차원에서 차단.
2. **§18 매트릭스 ID 갱신** — PR 2 머지 시점(2026-05-14) 에 환경변수 fallback 을 `gemini-3.0-flash` 로 정합 갱신했으므로, 매트릭스 본문 ID 동기화. §18-5 "키 교체 없이 업그레이드" 정합.
3. **§18 매트릭스 신규 행 2건** — AI 서술형 분석(이미 PR 2~5 동작 중)·챗봇(PR 6 신설) 모두 §18-6 "매트릭스 변경 = 헌법 개정" 정합으로 명시 등재.
