## 1. 발의 배경

PR #45 (v3.6.2) 머지 후 운영 시험 결과, 서술형 채점/첨삭(`gemini-2.5-pro`) 응답이 5~10초 소요되어 학습 흐름이 단절된다는 사용자 보고가 있었다 (2026-05-14 직접 발화).

> "ai 가 질문 답변에 너무 오래걸려 그냥 ai에 대한 모든 모델을 일단 flash로 바꾸고 속도를 최우선으로 하자"

본 발의로 §18 A 매트릭스의 *학습 본업* 3개 행(서술형 채점·첨삭·모범답안 생성)을 Pro → Flash 임시 통일한다. 추후 한국어 학술 평가 품질이 부족하면 Pro 재격상 검토.

### 2.3 개정 사유

1. **응답 시간 5~10초 → 1~2초** — 학습 흐름 단절 회피. 학습자가 답을 작성한 직후 분석 결과를 바로 받아 학습 가속.
2. **Pro 추론 깊이의 한계 효용** — 정성 피드백(강점·보완점)은 Flash 도 충분히 한국어 학술 평가 가능. Pro 의 깊이가 학습자가 인지하는 차이로 이어지지 않음.
3. **운영 비용 절감** — Flash 호출당 단가가 Pro 의 1/5~1/10 수준. 학습자 답안 분석은 카드당 1회 캐시 hit/miss 패턴이라 호출량 누적이 무시 못 함.
