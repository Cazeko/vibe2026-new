## 3. 결제 모델 [B4]

### 3.1. Pass 모델 정합 근거

- 임용 시험은 *1년 1회 (11월 말~12월 초)* — 월 정기결제는 사용자 멘탈 모델과 불일치
- 박문각·해커스 패스 (80~150만원) 가 한국 임용 시장 표준 가격 모델
- "내 시험일까지 fixed" 가 결제 결정의 심리적 안심 요인 (월결제 부담 회피)

### 3.2. 무료 + Pass 2단 구조

```
무료 사용자 (Free)
  ├─ 가입 + 시험일 입력 + 마이페이지
  ├─ 24년치 기출 분석 풀 노출 (히트맵·토픽맵·로드맵·연도별 분석)
  ├─ 풀이 트랙 SRS — 일 5장 제한 (시험 체험)
  ├─ 키워드/오답 트랙 — 잠금 (Pass 권유 안내)
  ├─ 팟캐스트 — 시드 라이브러리 N편 무료 (자동 생성 X)
  └─ 학습 계획 D-day 기본
      ↓ "Pass 결제 시 잠금 해제"
Pass 사용자 (49,000원, 시험일까지)
  ├─ 풀이/키워드/오답 3 트랙 SRS 무제한
  ├─ AI 모범답안 + 5탭 분석 (총평/키워드/diff/모범답안/이력) 무제한
  ├─ AI 학습 도우미 (튜터 챗봇) 무제한
  ├─ 팟캐스트 자동 생성 + 무제한 청취
  ├─ 답안 마크/필터, 큐 점프, 소문항 분리 입력 등 워크스페이스 풀
  └─ 학습 분석 (디테일 차트)
```

### 3.3. 결제·환불·갱신 정책

| 항목 | 정책 |
|---|---|
| 결제 시점 | 가입 시 즉시 결제 가능 (강제 X), 무료 체험 후 자유 전환 |
| 결제 수단 | 카드 + 토스페이 + 네이버페이 + 휴대폰 결제 |
| 환불 | 결제 후 **7일 무조건 환불** (사용 분량 무관) |
| 환불 (7일 후) | 사용 일수 비례 부분 환불 (시험일까지 잔여 일수 / 전체 가입 기간) |
| 자동 갱신 | **없음** — Pass 만료 후 자동 무료 다운그레이드, 재구매 시 사용자 명시 결제 |
| 시즌 갱신 | 시험 종료 후 익년 시즌 재구매 — 동일 가격 (충성 사용자 할인 검토 X, 정직성 §3의2 정합) |

4. **AI 분석 안정성** — Gemini Flash 응답 성공률 ≥ 99% (#71 hotfix 정합)
