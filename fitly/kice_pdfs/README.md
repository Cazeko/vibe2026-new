# kice_pdfs — 임용 기출 PDF 시드 자산

본 폴더는 fitly v3.1 시드 파이프라인의 입력 자산이다 (헌법 v3.1 부칙 제2조).

## 출처

- **한국교육과정평가원 (KICE)** 공식 공개 기출
- 본 자료는 국가공식 공개 자료이며, 사설 학원·교재의 무단 복제·인덱싱이 아니다 (헌법 제27조 정합)

## 파일명 규칙 (2026-05-06 정규화 완료)

**1차 시험 표준 형식**: `YYYY-{session}.pdf`

| session | 의미 |
|---------|------|
| `essay` | 교직논술 (1교시) |
| `A` | 교육과정 A (2교시) |
| `B` | 교육과정 B (3교시) |
| `combined` | 통합본 — 2009/2010/2011 (교시 분리되지 않은 옛 형식) |

예: `2025-essay.pdf`, `2025-A.pdf`, `2025-B.pdf`, `2009-combined.pdf`

## 수집 범위

| 연도 범위 | 교시 수 | 비고 |
|----------|--------|------|
| 2001~2008 | essay/A 또는 A만 | 객관식 시대 (분석·키워드 시드 대상, 풀이 카드 X) |
| 2009~2011 | combined | 옛 통합 형식 — 운영자 수동 분리 대상 (plan Appendix A #7) |
| 2013~2026 | essay/A/B | 표준 1차 형식 (2014~ 서술형 풀이 카드 대상) |

> 2012는 누락 (시험 폐지/미시행).

## _archive/ — 시드 파이프라인 제외 자료

| 폴더 | 내용 | 사유 |
|------|------|------|
| `_archive/secondary/` | 2차 시험 자료 (`*-2nd-*.pdf`) | v3.1 1차 시험만 다룸 |
| `_archive/answers/` | 정답표 PDF | 사용자 결정 (2026-05-06) — 시드 무시 |
| `_archive/uncategorized/` | 출처/형식 불명 자료 | 운영자 검토 필요 |

## .gitignore 정책

본 폴더의 PDF는 git에 포함되지 아니한다 (큰 용량 회피). 본 README만 트래킹된다.

신규 환경에서 fitly 시드 파이프라인을 실행하려면 KICE 공식 사이트에서 위 범위의 PDF를 다운로드하여 본 폴더에 배치한 후 `scripts/seed/01-normalize-papers.ts` 또는 동등한 정규화 스크립트를 적용한다.
