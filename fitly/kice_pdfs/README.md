# kice_pdfs — 임용 기출 PDF 시드 자산

본 폴더는 fitly v3.0 시드 파이프라인의 입력 자산이다 (헌법 v3.0 부칙 제2조).

## 출처

- **한국교육과정평가원 (KICE)** 공식 공개 기출
- 본 자료는 국가공식 공개 자료이며, 사설 학원·교재의 무단 복제·인덱싱이 아니다 (헌법 제27조 정합)

## 수집 범위

- 기간: 2002~2026 (24년치)
- 시험: 초등학교교사 임용후보자 선정경쟁시험 1차
- 교시: 교직논술 / 교육과정 A / 교육과정 B
- 분량: 약 50~68개 PDF

## 파일명 규칙 (정규화 입력)

`scripts/seed/01-normalize-papers.ts` 가 다음 패턴을 인식한다.

| 패턴 예시 | 추출 |
|----------|------|
| `2024학년도_초등학교_교직논술.pdf` | year=2024, session=논술 |
| `2017학년도 초등학교 교육과정_A.pdf` | year=2017, session=A |
| `2011초등2차-초등 교육과정A.pdf` | (제외 — 2차 시험) |
| `*정답*.pdf` | (제외 — 정답표) |

## 제외 대상

- 정답표 PDF (사용자 결정, 2026-05-06)
- "초등2차" 시리즈 (1차 시험만 다룸)
- 2009/2010 통합본 (스킵 또는 수동 분리 — 운영자 처리)

## .gitignore 정책

본 폴더의 PDF는 git에 포함되지 아니한다 (큰 용량 회피). 본 README만 트래킹된다.

신규 환경에서 fitly 시드 파이프라인을 실행하려면 KICE 공식 사이트에서 위 범위의 PDF를 다운로드하여 본 폴더에 배치한다.
